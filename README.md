O RHPointer é um sistema de gerenciamento de ponto eletrônico desenvolvido em php, de forma que pode ser usado para trackear as horas trabalhadas


### Tecnologias

Dentre as tecnologias usadas, temos:

- Jquery
- Javascript
- HTML/CSS
- PHP
- Composer
- Docker
- Apache
- MySQL
